# AXUAURA BOT
---

This is a Discord bot written entirely in C++ using the D++ library.

This bot was custom created for a specific server, but work has been done to push it towards cross server modularity. At this time, it is about 80% compatible across servers. 

No assistance will be offered in self hosting the bot, but a public version will be made once 100% feature parity is achieved across servers. The bot has very low requirements for those who do decide to self host it, through testing, it idles at approximately 30MiB of RAM.

---

## Features

- Full economy system using currency of aura, gained by chatting and lost by doing "auraless things" like failed embeds. The currency features two sides (the negative and the positive) where users can either aim to have as little aura as possible, or as much as possible. 
- Gambling and duels using above currency, featuring animations using gifs.
- Fully customizable shop for custom role colours, level up banners (server specific) and more using the aforementioned aura currency.
- Level/XP system similar to other established bots, but featuring purchaseable XP boosters from the aura shop.
- Custom banners and other assets drawn by artist @axuaxi. Levels feature growing tree backgrounds by default, but users are able to equip custom banners if they wish.
- Starboard variation allowing users to pay aura to boost a post. 
- And much more!
