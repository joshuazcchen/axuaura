# THE AXUAURA BOT
---

Tentatively this is a private discord bot, but it will be made public in the near future.


This bot was made for the Axuaxilings discord server, and has since grown to be a fully fledged level and economy bot.


No support will be offered in self hosting the bot, but a public instance will be made available soon. If you wish to self host the bot, you are acknowledging that there can/will be errors and the bot may not work as intended at times. 


This bot requires G++16 (or any other C++ compiler capable of C++26 or higher, due to the usage of `std::philox_engine`), which is a higher version than included on default by Ubuntu and many other distros.
